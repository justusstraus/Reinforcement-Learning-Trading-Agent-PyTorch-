# Reinforcement-Learning-Trading-Agent-PyTorch-

Overview

This project implements a reinforcement learning trading agent in PyTorch designed to trade oil and gold futures. The agent learns an optimal trading policy by maximizing a Sharpe-ratio-based reward function, encouraging strategies that achieve high risk-adjusted returns rather than raw profit.

The project explores how reinforcement learning can be applied to systematic trading strategies in financial markets, combining financial time-series data with deep reinforcement learning techniques.
